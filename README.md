# TotalCross CLI POC

POC for P&C activity

## Usage

1. Install dependencies

``` 
$ npm install
```

2. Run main script

``` 
$ node index.js --create
```

3. Test another commands

``` 
$ node index.js --pack
```
and: 

``` 
$ node index.js --deploy
```
